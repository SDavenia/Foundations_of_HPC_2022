First of all we need to allocate an EPYC or a thin machine.
salloc -n1 -N1 -p EPYC --time=0:15:00

Then you need to load an architecture, when working with EPYC nodes we need AMD architecture
module load architecture/AMD
module load module load openMPI/4.1.4/gnu/12.2.1

srun mpicc MPI_hello_world.c -o hello_world.x

Alternatively we can first compile and then use srun to be allocated a new machine
mpicc MPI_hello_world.c -o hello_world.x (to compile we do not need to have allocated resources)
srun hello_world.x -N1 -n4 -p EPYC --time=0:15:00 --pty bash

To do the same using C++ you can work with mpiCC instead of mpicc for compilation, all the rest is the same.
